# Appliance Repair Landing Page

## Overview
A simple responsive landing page for a U.S. home appliance repair company.

## Features
- Header with navigation
- Services section
- Booking form with validation
- Responsive layout
- Footer with contact information

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Structure
- index.html
- style.css
- script.js

## How to Run
Open `index.html` in a browser or run the project with Live Server in VSCode.
